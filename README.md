# Student Data Website

A simple multi-page personal portfolio website built using pure HTML as part of the IIT Madras BS in Data Science coursework. The website demonstrates the fundamentals of HTML by creating interconnected webpages with navigation, tables, images, hyperlinks, downloadable documents, and semantic HTML elements.

---

## Features

* Multi-page website with navigation bar
* Home page with student details
* Academics page displaying marks in a table
* Personal page with images and external website link
* Contact page using semantic `<address>` tag
* Resume page with downloadable PDF
* Clean HTML structure using semantic elements
